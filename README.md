# LB-Chat

通过互联网和朋友聊天吧。

## 特点

浏览器无需 JavaScript 支持即可在线网页聊天。

## 服务端

文件：`./server/server.py`

运行：`python server.py --port 2666`

## 客户端

直接运行：`curl -sS https://gitee.com/PJ-568/lb-chat/raw/main/client/client.sh | bash`

文件：`./client/client.sh`

运行：`bash ./client.sh --cli --dialog --zenity --help --version --zh --en`
